# 커뮤니티 기여 (Community Contributions)

## 기여하는 방법 (How to Contribute)
여러분의 튜토리얼이나 SDK를 이곳에 등록하고 싶으신가요? 작업하신 결과물의 링크를 `pubgapi@pubg.com`으로 이메일을 보내주세요!

## 더 이상 유지보수되지 않음 (No Longer Maintained)
