# 소개 (Introduction)

## 환영합니다 (Welcome)
PUBG 게임 데이터 서비스(Game Data Service)는 사용자가 인게임 데이터에 개방적으로 접근할 수 있도록 지원하며, 개발자는 이를 활용하여 멋진 도구와 서비스를 구축할 수 있습니다.

## JSON API
이 서비스는 JSON-API 사양의 필수 기능을 구현하도록 설계되었습니다. 각 API 엔드포인트에서 반환되는 데이터는 JSON-API 형식을 따릅니다. 이 사양에 대한 추가 정보는 해당 공식 문서에서 확인할 수 있습니다.

## OpenAPI
OpenAPI 사양을 사용하여 API 엔드포인트 문서를 작성했습니다. 즉, 사용자는 기술적 경험이나 코딩 지식이 전혀 없어도 엔드포인트 문서와 상호작용하여 실제 작동 방식을 확인하고, 명령을 생성하며, 반환되는 데이터의 형태를 파악할 수 있습니다! 엔드포인트 문서 페이지에서 자물쇠 기호가 있는 녹색 "Authorize(승인)" 버튼을 클릭하고 API 키를 입력하세요. 완료되면 아무 엔드포인트에서나 "Try it out(사용해 보기)"을 클릭하고 "Execute(실행)" 버튼을 사용하여 요청을 보낼 수 있습니다.

## 에셋, 데이터 사전 및 열거형 찾기 (Finding Assets, Data Dictionaries, and Enums)
데이터 사전 및 열거형(Enums)은 GitHub에서 찾을 수 있습니다.

## 도움말 얻기 (Getting Help)
자주 묻는 일반적인 질문에 대한 답변은 FAQ를 살펴보는 것을 권장합니다. 또한 PUBG Developer API Discord 서버에 참여하여 다른 커뮤니티 개발자 및 디자이너와 만나고, 소통하고, 문제를 해결하며, 협업할 수 있습니다 (단, 공식 운영 서버는 아닙니다).
